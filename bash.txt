`bash`

ls - list storage 

pwd - present working directory 

ls -a показать всё
ls -l лонг лист
ls -la лонг лист всего

cd .. - вернуться назад
cd - возвращает в начало
~ - возвращает в начало
pushd - закидывает в новую директорию с
возможностью вернуться в прошлую
popd - возвращает в старую директорию

file . - информация о файле
locate - поиск файла
which - найти комманду
history - история
whatis - заменяет гугл, объясняет за комманды
apropos - ручной поиск по именам страниц и описаний
man - показывает мануал
mkdir - создать директорию

touch - создаёт временные файлы
cp - копирует файлы и директории
mv - двигает (переименовывает) файл
rm - удаляет файлы и директории
cat - конкатенирует (сцепляет) файлы и печатает их
more - разделяет большой файл на несколько страниц
less - обратное more
q - выход
nano - редактор

cat >> - добавить текст
cat > - заменить текст
ls -al / > - записать весь текст из папки в один файл
sudo - superuser do, запуск от имени админа
su - создать нового пользователя
chmod - показывает тип файла в битах
chmod +x - сделать файл executable

watch - наблюдение каждые 2 с.
watch free -h - посмотреть память
killall - выключить прогу
